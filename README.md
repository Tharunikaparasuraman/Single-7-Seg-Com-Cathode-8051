# Single 7-Segment Common Cathode (8051)

## Description
This project demonstrates how to interface a Single 7-Segment Common Cathode display with the 8051 microcontroller. The program is written in Embedded C using Keil uVision and simulated in Proteus. The display is used to show numeric digits by sending the corresponding segment patterns from the microcontroller.

## Tools Used
- Keil uVision
- Proteus

## Components
- AT89C51 Microcontroller
- Single 7-Segment Common Cathode Display
- Resistors
- Crystal Oscillator
- Capacitors
- Power Supply

## Files
- `single 7 seg common cathode.c` – Embedded C source code
- `single 7 seg common cathode.hex` – Compiled HEX file
- `single 7 seg common cathode.uvproj` – Keil project
- `single 7 seg common cathode.pdsprj` – Proteus simulation project
- `proteus connection.png` – Circuit connection screenshot

## Output
The Common Cathode 7-Segment display shows the programmed numeric digits based on the Embedded C program.
